One probabilistic pass

Content and structure come out together, fused, in a single pass.

For a static one-off, that’s fine.

The moment you want to change something, you’re asking a probabilistic system to edit a fused artifact — and hoping it touches only what you meant.

A small wink is fine here. Set up the fragility without naming Quarto yet.

Quarto in the middle
